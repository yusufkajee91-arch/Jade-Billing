New Search Help

This header search is now a search-engine style search across matters, clients, suppliers, invoices, collection debtors and more. Use quotatation marks to do phrase searches, minus (-) to search by exclusion of keyword.

Shortcut Key Tip: You can activate the header search with the shortcut key forwardslash / (whereas backslash \ focuses the menu filter search on the left).

Header Search Help

Just start typing to search for matters. Enter all or some of the letters of a matter or client to search for a matter. Eg nkosinathi

Searches of other items are possible by preceding the search with a keyword and a space. Examples:

To search for an invoice, type invoice followed by space and then part of the invoice number, reference or amount. Eg invoice 1001

You can also use words like today or yesterday or specify a date in ISO format (YYYY-MM-DD)af. Use > to indicate that you want to create a new record, or . to indicate you want to navigate to a list of records. Here are some example searches to find records and transactions:

  • receipt 1000
  • client acme
  • supplier woolworths
  • account telephone
  • invoice today
  • receipt yesterday
  • supplier >(to create new supplier)
  • client .(to go to a tabular list of clients)

Shortcut Key Tip: You can activate the header search with the shortcut key forwardslash / (whereas backslash \ focuses the menu filter search on the left).
